Readme File for Vincent La (vla6) for manual_strategy project

To run this code, simply go into the `manual_strategy` folder with the correct python environments.

You should be able to just run the code by typing `python <filename.py>`

For example, to run:
```
$ python indicators.py
$ python BestPossibleStrategy.py
$ python ManualStrategy.py
```

In addition, the file `marketsimcode.py` while doesn't need to run directly, contains the relevant helper
functions to compute the portfolio values.
